현재 Apple Sum 10 Multiplayer 프로젝트에 팀전 모드를 추가한다.

대기실에서 개인전/팀전을 전환할 수 있어야 하고, 팀전에서는 팀 수 설정, 랜덤 팀 배정, 방장의 수동 팀 배정이 가능해야 한다. 팀전은 개별 화면 모드와 단일 화면 모드를 모두 지원해야 한다.
개별 화면 모드는 각 플레이어가 기존처럼 자신의 화면에서 개별적으로 플레이한 뒤, 라운드 종료 시 플레이어별 점수를 팀별로 합산하는 방식이어야 한다. 팀전에서는 팀 점수를 중심으로 리더보드를 표시하되, 필요하면 팀 내부의 개인 점수도 함께 확인할 수 있어야 한다.

단일 화면 모드는 같은 팀이 하나의 공용 보드를 공유하는 방식이어야 한다. 같은 팀원은 동일한 사과 보드를 보아야 하며, 팀원의 포인터 이동과 사과 제거 결과가 Firebase Realtime Database를 통해 실시간으로 서로의 화면에 반영되어야 한다. 사과 제거는 중복 적용되지 않아야 하며, 동시에 같은 사과를 제거하려는 상황에서도 팀 공용 점수와 보드 상태가 깨지지 않아야 한다.
기존 Firebase Realtime Database 기반 멀티플레이 구조, 라운드 진행, 채팅, 재접속 흐름은 유지하면서 단계적으로 구현한다. 구현 순서는 로비 팀전 설정과 팀 배정, 개별 화면 팀전 리더보드, 단일 화면 팀전 공용 보드 상태, 팀 포인터 동기화, 충돌 처리와 재접속 및 Firebase rules 보강 순서로 진행한다. 오래 분석하고 단계적으로 설계하라.

종료 시점은 다음 조건을 모두 만족했을 때로 한다.
1. 개인전 기존 기능이 깨지지 않고 정상 동작한다.
2. 대기실에서 개인전/팀전 전환, 팀 수 설정, 랜덤 팀 배정, 방장 수동 팀 배정이 정상 동작한다.
3. 개별 화면 팀전에서 각 플레이어의 점수가 라운드 종료 후 팀별로 정확히 합산된다.
4. 개별 화면 팀전 리더보드에서 팀 순위와 팀별 점수가 정상 표시된다.
5. 단일 화면 팀전에서 같은 팀 플레이어들이 하나의 공용 보드를 공유한다.
6. 단일 화면 팀전에서 팀원의 포인터 위치가 실시간으로 표시된다.
7. 단일 화면 팀전에서 한 팀원이 제거한 사과가 다른 팀원 화면에도 즉시 반영된다.
8. 같은 사과를 여러 명이 거의 동시에 제거해도 점수가 중복 반영되지 않는다.
9. 라운드 진행, 채팅, 재접속, 리더보드, 방장 권한 흐름이 팀전에서도 기존 개인전과 충돌 없이 동작한다.
10. Firebase Realtime Database에 저장되는 팀 정보, 팀 배정, 팀 점수, 공용 보드 상태, 포인터 상태가 재접속 후에도 복구된다.
11. Firebase rules가 팀전 데이터 구조에 맞게 보강되어 잘못된 팀 데이터 수정이나 비정상적인 점수 조작을 막는다.
12. 서로 다른 브라우저 탭, 서로 다른 기기, 느린 네트워크, 새로고침, 중간 퇴장, 재접속 상황에서도 팀전 상태가 가능한 한 일관되게 유지된다.
13. 최소 2명부터 가능한 최대 인원까지 팀 배정과 팀 점수 계산이 정상 동작한다.
14. 팀 수를 변경하거나 팀원을 수동 이동한 직후 게임을 시작해도 잘못된 팀 배정이 남지 않는다.
15. 단일 화면 팀전에서 포인터 이동, 사과 제거, 점수 반영, 보드 갱신이 모두 실시간으로 보이며 체감상 지연이 크지 않다.
16. 기존 local fallback 또는 Firebase 연결 실패 시나리오에서도 앱이 완전히 깨지지 않고 가능한 범위에서 안전하게 동작한다.
17. 팀전 관련 타입, 상태 모델, Firebase read/write 함수, UI 컴포넌트가 중복 구현 없이 정리되어 있다.
18. 구현 후 TypeScript 타입 오류, lint 오류, 빌드 오류가 없어야 한다.
19. 방장, 일반 플레이어, 재접속 플레이어, 중간 입장 플레이어 관점에서 모두 수동 테스트가 완료되어야 한다.
20. 아주 이상한 종료 조건으로, 단일 화면 팀전에서 팀원 4명이 동시에 같은 사과 묶음을 클릭하고, 그중 1명이 새로고침하고, 1명이 네트워크가 잠시 끊기고, 방장이 같은 순간 채팅을 보내고, 라운드가 거의 동시에 종료되는 상황에서도 점수, 제거된 사과, 포인터, 리더보드, 재접속 상태가 최대한 논리적으로 복구되어야 한다.
